# ProjetofinalTeoriadosgrafos
Projeto desenvolvido para a disciplina de Teoria dos Grafos com o objetivo de modelar uma rede social profissional utilizando grafos. Os usuários são representados por vértices e as conexões por arestas. A aplicação implementa a sugestão de conexões com base em amigos em comum, aplicando conceitos de análise de grafos em Java.
